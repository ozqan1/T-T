# TİT 🦂 — Akrep İndirici

Nebi Özkan tarafından geliştirilen, **X (Twitter) / TikTok / Instagram** video indirme uygulaması.

> Ücretsiz ve reklamsız video indirme. Parmak izi/yüz tanıma kilidi, gömülü oynatıcı ve kişiselleştirilebilir temalar.

Bu depo, React Native (Expo) ile geliştirilmiştir. APK, GitHub Actions ile otomatik derlenir.
